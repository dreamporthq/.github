# DreamPort

![Desktop](https://img.shields.io/badge/Desktop-public-blue)
![iOS](https://img.shields.io/badge/iOS-App%20Store%20planned-lightgrey)
![Status](https://img.shields.io/badge/Status-Active%20Development-brightgreen)

A cross-platform app that converts Apple Watch sleep stage data from Apple Health into an OSCAR-compatible import format.

---

## Overview

DreamPort is designed to make Apple Watch sleep stage data easier to use with OSCAR.

## Public Repositories

- [dreamport](https://github.com/dreamporthq/dreamport) — Overview, public information, and project hub
- [dreamport-desktop](https://github.com/dreamporthq/dreamport-desktop) — Desktop app repository

## Availability

- **iOS:** Coming soon on the App Store
